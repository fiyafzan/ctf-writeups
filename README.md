# CTF Write-ups

A personal collection of write-ups from Capture The Flag (CTF) competitions I've participated in, covering web exploitation and other categories as they come up.

## Structure

Each folder corresponds to one CTF event. Inside, you'll find the write-up(s) for the challenge(s) solved during that competition.

```
ctf-writeups/
├── 0xCYNX-Live-CTF/
│   └── Zer0D4Ys-Internal-Affairs.md
├── (next CTF event)/
│   └── ...
```

## Disclaimer

All challenges documented here were solved in controlled, intentionally-vulnerable CTF/training environments. None of this involves real-world systems or unauthorized access.
